Lesson 7: Negatives

The Core of Negatives: ない

ない is an adjective meaning "non-existent / not-be".
When we don't do something, we describe a state of non-action → adjective!
Positive Meaning Negative Meaning
ある exists (things) ない doesn't exist
ペンがある There is a pen ペンがない There is no pen

The A-Stem (Negative Stem)

Verb Negative: A-stem + ない
Shift the final kana from う-row to あ-row, then add ない.
Verb A-stem Negative
聞く (く) 聞か 聞かない
話す (す) 話さ 話さない
持つ (つ) 持た 持たない
飲む (む) 飲ま 飲まない
走る (る) 走ら 走らない
Exception: う → わ (not あ)
買う → 買わない (NOT 買あない)
This is the ONLY exception in the entire stem system!

Ichidan Verbs - Simple as Always

食べる → 食べない

Just drop る, add ない

Adjective Negatives

い-Adjective Negative: Drop い → Add くない
Adjective Negative Past Negative
赤い 赤くない 赤くなかった
怖い 怖くない 怖くなかった

Noun Negative (This is not X)

Pattern: noun + ではない

これはペンではない

This is not a pen. (で = て-form of だ)

Past Negative

ない is an い-adjective, so past = なかった
Form Non-past Past
Positive 走る 走った
Negative 走らない 走らなかった

Exceptions

Word Issue Correct Form
ます Uses old negative せん ません (not ますない)
いい Reverts to よい よくない / よかった
いい changes back to よい!
✗ いくない, いかった
✓ よくない, よかった

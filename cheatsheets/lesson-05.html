Lesson 5: Verb Groups & て/た Forms

The Three Verb Groups

Group Name Endings Examples
1 Ichidan (一段) -いる, -える only 食べる、見る、寝る
2 Godan (五段) Any う-sound 買う、書く、話す
3 Irregular Only 2 verbs! 来る、する

Ichidan Verbs - Always Simple

Just drop る and add whatever you need. That's it!
Verb て-form た-form
食べる 食べて 食べた
見る 見て 見た

Godan Verbs - Four Patterns

1. UTUSURU Group (う, つ, る)

"Utusuru" (移る) = to move. Has つ in middle → uses っ
Ending→ て→ たExample
うってった笑う → 笑って/笑った
つってった持つ → 持って/持った
るってった取る → 取って/取った

2. NEW BOOM Group (ぬ, ぶ, む) - Dull Sounds

Dull sounds (ぬ,ぶ,む) → Dull ending (んで/んだ)
Ending→ て→ たExample
ぬんでんだ死ぬ → 死んで/死んだ
ぶんでんだ遊ぶ → 遊んで/遊んだ
むんでんだ飲む → 飲んで/飲んだ

3. く/ぐ Group

く → いて | ぐ → いで (just add dakuten)
Ending→ て→ たExample
くいていた歩く → 歩いて/歩いた
ぐいでいだ泳ぐ → 泳いで/泳いだ

4. す Group

Ending→ て→ たExample
すしてした話す → 話して/話した

Exceptions (Only 3!)

Verb て-form た-form Note
来る 来て (きて) 来た (きた) Irregular
する して した Irregular
行く 行って 行った Not 行いて!
Watch out for 行く!
行く looks like く-group but uses って/った like utusuru group.
